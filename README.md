# BACK FOLDER
```
cd back
```

### Create .env file containing these informations
```
# Secret token for authorization
- SECRET=""

# Secret admin password
- ADMIN_PASSWORD=""

# MySQL database informations
- DB_HOST=""
- DB_NAME=""
- DB_USERNAME=""
- DB_PASSWORD=""
```

### Setup
```
npm install
```

### Launch
```
npm start OR nodemon server
```
